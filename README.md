# うきまる – 美しい世界

> **歌詞登録ステータス：プレーン歌詞のみ**

```
かじかんだ手で掴んだものは
いつもすぐに溶けて無くなってしまう
会えない日々が寂しいのは
あなたのことを知ってるから
今にも溢れ出てしまうような
言葉たちを飲み込んで
冴えない日々に花を添えていけるように
涙無しで生きるには少し冷たい
期待しても何も訪れなくて
沈む夕日を隠す厚い雲のように
見えなくなってしまう 美しい世界
遠く離れている小さな光
いつか会えるかな また会えるかな
沈む夕日を映す広い海のように
見えなくなってしまう あなたの瞳
遠く離れている小さな光
いつか会えるかな また会えるかな
```
